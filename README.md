### 简介

此项目是学习 PWA 的项目，记录学习的点滴，方便以后忘记可以从学习轨迹中迅速上手。

## 如何开发

```
// 安装依赖
npm ci

// eslint 规范
npm run lint

// test 测试用例
npm run test
```
